## Files

- `Movies Recommendation System.ipynb` - Jupyter notebook with the main analysis and model
- `app.py` - Python application file
- `movie_list.pkl` - Serialized movie data
- `similarity.pkl` - Precomputed similarity matrix
- `tmdb_5000_credits.csv` - Movie credits dataset
- `tmdb_5000_movies.csv` - Movie details dataset

## Usage

1. Run the Jupyter notebook to see the analysis
2. Use `app.py` for the recommendation application

## Dataset

This project uses the TMDB 5000 Movie Dataset containing movie information and credits.
